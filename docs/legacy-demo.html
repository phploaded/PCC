PCC stands for Project Cost Calculator. Below of the page, you have a demo forms for web developent company, Party Cake Bakery and a demo of custom PC builder. Please note that you dont have to copy exactly same forms given here, but using this jQuery plugin, you can build your own forms as you like, with your own options, features and costs. You can have multiple forms at same page.

NEW : You can change the currency and green button text.
NEW : You can charge based on per item now ( item_cost X total_items = total_for_the_item ).
NEW : You can create textboxes and textareas to get user information, they will only appear in email.
